Installs the Blue Sky Chat platform on to your website. 

== Description ==

Blue Sky Chat is an <a href="https://www.blueskychat.com">Outsourced Live Chat</a> lead generator based in Australia. The plugin allows you to easily install and activate the Blue Sky Chat platform on to your website without the need to enter or upload any manual codes.

Please note that in order to properly use this service, an active subscription will be required which can be obtained from https://www.blueskychat.com

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Do I need a subscription plan for this to work? =

The plugin will install without a subscription plan, however for the service to be active you will need to head over to <a href="https://www.blueskychat.com" target="_self">Blue Sky Chat</a> to make a purchase.
